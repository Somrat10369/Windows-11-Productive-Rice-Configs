

## ⚙️ Essential Configuration

After installation, run these configurations for optimal experience:

### Must-Do Configurations

```powershell
# 1. Set default terminal
Set-DefaultTerminal -Name "Terax"

# 2. Configure Flow Launcher startup
Set-StartupApp -Name "Flow Launcher" -Enable

# 3. Disable startup programs
Set-StartupManager -Tool "HiBit" -Action "Enable"

# 4. Install Everything + Toolbar
Set-Everything -Action "Install"

# 5. Configure VS Code
cd "C:\Essantials\Documents\Code\Win 11 Config\Vscode"
# Edit settings.json as needed
```

### Recommended Settings

**Terminal:**
- Set Terax as default
- Enable font ligatures
- Configure theme (Oh-My-Posh)
- Enable shell history

**PowerShell:**
- Profile: Set custom profile in `PowerShell_Profile — Command Cheatsheet.md`
- Enable 7.x version
- Configure aliases
- Enable auto-complete

**Startup:**
- Use HiBit to disable unnecessary startup items
- Enable Flow Launcher for app launching
- Configure QuickStartup for boot optimization

**Visual:**
- Apply Catppuccin theme
- Install Tacky-Borders for focus indicators
- Set YASB Reborn for status bar
- Configure Fluent Weather for weather
