# Custom UI Mods for Vivaldi

⚠️ **ATTENTION:** To use custom UI mods, you _must_ enable experimental CSS modification flags in Vivaldi first.

## ⚙️ Setup Instructions

Please follow these steps in order to successfully apply your custom theme:

### Step 1: Enable Experimental Flag

Navigate directly to the following internal page in Vivaldi:
[vivaldi://flags](vivaldi://flags)

Search for **"Allow CSS modifications"** and change its setting from `Default` to **`Allow`**. Remember to restart your browser after changing this flag.

### Step 2: Select Custom Folder

1. Go to **Settings** > **Appearance**.
2. Locate the **"Custom UI Modifications"** option.
3. Click **"Select Folder"** and point it to the directory containing all your custom styles for this mod.

---

_After completing these steps, Vivaldi should be able to load your custom interface._
